# 🌐 Personal Portfolio

This is my personal developer portfolio built using **React** and **Tailwind CSS**.
It showcases my projects, skills, and provides a way to download my resume.

🔗 **Live Demo:**
https://portfolio-sage-three-21.vercel.app/

---

## 🚀 Features

* Responsive modern UI
* Projects showcase
* Skills section
* About section
* Contact section
* Resume download
* Clean and minimal design
* Fast build using Vite

---

## 🛠️ Tech Stack

* **React.js**
* **Tailwind CSS**
* **Vite**
* **JavaScript**
* **HTML5**
* **CSS3**

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/manavv09/Canvas.git
```

Go to project folder

```
cd Canvas
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```
